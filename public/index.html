Web Project

Professor might want to build courses taking resources from various sources like video links and static web pages. There is currently no such system out there. We are trying to address this by making a web portal where professors can hand pick course material from different sources (static web pages, video links) and also associate skills that can be acquired during the course. He can then make them as a part of his module. All the courses are then made available to the student and he can choose his course according to the skills he wants build on. A course is modeled in such a way that no student can change the order of the particular course. The professor decides the flow of the course and that is only way in which the course will be delivered by the website. At no point the student can skip a mandatory module. This ensures that the course has been taken in the right manner. A student can also go to the profile page and see all the skills he has acquired from the modules/units he has completed so far.
